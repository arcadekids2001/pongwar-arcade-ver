# PongWAR! Arcade Ver.2

PongWAR! is a small arcade game that mixes classic Pong with Spacewar-inspired shooting. You play against a stubborn AI paddle, but you can shoot to chip away at its health and turn the match around.

## Play

- Move the mouse to control the player paddle.
- Press `Space` to shoot.
- Keep your health above zero.
- Reduce the AI health to zero to win.

## Run Locally

Open `index.html` in a browser.

No build step is required. The game uses Vue from a CDN.

## Publish With GitHub Pages

After uploading this folder to GitHub:

1. Open the repository on GitHub.
2. Go to `Settings` -> `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Choose the `main` branch and `/ (root)`.
5. Save.

GitHub will give you a public link when the page is deployed.
